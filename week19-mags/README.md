## Learning objectives

- Assess MAG quality (completeness, contamination)
- Filter MAGs by quality thresholds
- Assign taxonomy using reference databases
- Annotate genes and proteins
- Identify CAZymes
- Detect biosynthetic gene clusters (BGCs)
- Map annotations to metabolic pathways
